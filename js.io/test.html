what is js.io?

The js.io library at its core is a collection of open protocol implementations and tools for writing socket-based applications for the web.

how do i use js.io?

To use js.io, simply import js.io.js with a <script> tag in the head of your html. At that point, you have access to all the functionality js.io has to offer with js.io.require. In development, "js.io.require( 'js.io.protocols.websocket' )" will see if the websocket protocol is available, and if not, will make an xhr request for websocket.js. In deployment mode, the client will get all necessary functionality in one initial request for a compiled javascript project file.
